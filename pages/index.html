This game on the Cardano blockchain is a simple image-generating service that is powered by the PIGY Token. Sending the PIGY Token and ADA to the address above will either mint, blend, or burn Cardano native tokens, each with a customized image of a pig.

The images can be viewed by visiting the IPFS link in the minting metadata for the token, or by visiting a token-viewing explorer such as pool.pm/tokens.

The image-blending process randomly combines attributes of the "parent" images to create a new "child" image. Carefully chosen blending strategies can produce "offspring" with particular target characteristics. The diagram to the upper right shows the "family tree" for the first pig-image tokens created using this service.

If no PIGY or if too little ADA is sent in the transaction, the service will not automatically mint, blend, or burn the pig-image token; nevertheless, the funds will be returned to the sending address via a manually-constructed transaction, in which case there may be a considerable delay (hours or days) in receiving them back. If you fail to receive the pig-image token, please check the metadata in the refund transaction to learn why it failed.

Work is in progress to add facial expression, clothing, jewelry, badges, and other accoutrements to the pig images. Eventually, this service will be migrated to a smart contract.

  • If less ADA than the minimum specified in the table above is provided or if no PIGY is provided, then no image token will be sent back.
